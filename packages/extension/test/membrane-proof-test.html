Membrane Proof Test

Tests the deferred membrane proof flow with a progenitor-gated test hApp. The DNA requires a membrane proof: the authorizer's Ed25519 signature of the installing agent's 39-byte AgentPubKey.

Instructions

Authorizer Setup (one-time, in the Lair popup)

  1. Open the HWC extension popup, go to Lair Keystore
  2. Under Seed Phrase Restore, enter tag: authorizer
  3. Paste these 24 seed words:
indoor clutch chicken nurse inhale close feel type school sweet oyster property balcony city actual erupt before stomach ethics talk pact camera lonely proud

This restores the authorizer key with Ed25519 pubkey: c41YjAiTC4HIGu85yZaCLyKDFhYArTFZmRqYSChrzHY=

Test Flow

  1. Detect extension and authorize domain (steps 1-2 below)
  2. Install the membrane-proof hApp (step 3) -- app enters "awaiting memproofs" state
  3. Copy the agent's AgentPubKey (shown after install)
  4. In the Lair popup Sign section: select authorizer key, set Data Format to Base64 (raw bytes), paste the AgentPubKey (base64) value, and sign
  5. Paste the signature as the membrane proof (step 4) and submit
  6. App should transition to "enabled" state
