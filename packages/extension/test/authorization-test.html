Testing Instructions

  1. Load the Holochain Web Conductor extension as an unpacked extension in Chrome
  2. Open this HTML file in your browser (either via file:// or localhost)
  3. Click "Test Connection" - should open authorization popup on first attempt
  4. Approve the connection in the popup
  5. Click "Test Connection" again - should succeed instantly without popup
  6. Open extension popup and go to "Manage Permissions"
  7. Revoke permission for this origin
  8. Click "Test Connection" again - should open popup again
