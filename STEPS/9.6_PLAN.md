# Plan: Step 9.6 - Wire Up Kitsune2 and E2E Remote Signal Test

## Discovery: Infrastructure Already Exists

After exploring the codebase, I discovered that **kitsune2 integration is already 90% built** in hc-http-gw-fork:

**Existing modules:**
- `src/kitsune_proxy.rs` - `KitsuneProxy` implements `KitsuneHandler`, `ProxySpaceHandler` handles `recv_notify()`
- `src/proxy_agent.rs` - `ProxyAgent` represents browser agents in kitsune2
- `src/agent_proxy.rs` - `AgentProxyManager` routes signals to WebSocket connections
- Router accepts `Option<GatewayKitsune>` parameter

**What's missing:** The `GatewayKitsune` is **never initialized** - it's always `None` in the binary startup.

### Step 3: Initialize GatewayKitsune in main()

**File:** `hc-http-gw-fork/src/bin/hc-http-gw.rs` (lines 38-60)

Build `GatewayKitsune` when enabled:

```rust
// Create GatewayKitsune if enabled
let gateway_kitsune = if configuration.kitsune2.enabled {
    let handler = KitsuneProxy::new(agent_proxy.clone());
    let mut builder = KitsuneProxyBuilder::new(handler);

    if let Some(url) = &configuration.kitsune2.bootstrap_url {
        builder = builder.with_bootstrap_url(url);
    }
    if let Some(url) = &configuration.kitsune2.signal_url {
        builder = builder.with_signal_url(url);
    }

    let kitsune = builder.build().await
        .context("Failed to build kitsune2 instance")?;
    Some(GatewayKitsune::new(kitsune))
} else {
    None
};

// Pass to service
let service = HcHttpGatewayService::with_auth(
    args.address,
    args.port,
    configuration,
    admin_call,
    app_call,
    None,
    Some(agent_proxy),
    gateway_kitsune,  // <- Add this parameter
).await?;
```

```rust
#[tokio::test]
async fn test_remote_signal_from_conductor_to_browser() {
    // 1. Set up test infrastructure
    let bootstrap = start_local_bootstrap().await;

    // 2. Start conductor with Agent A
    let conductor = SweetConductor::from_config(config_with_bootstrap(&bootstrap)).await;
    let (cell_a, agent_a) = conductor.setup_app("test", &[dna]).await;

    // 3. Start gateway with kitsune2
    let gateway = start_gateway_with_kitsune(&bootstrap).await;

    // 4. Connect WebSocket client, register Agent B
    let ws_client = connect_ws(&gateway).await;
    ws_client.send(Register { dna_hash, agent_pubkey: agent_b }).await;

    // 5. Agent A sends remote signal to Agent B
    let payload = encode_signal("Hello from conductor!");
    cell_a.call_zome("send_signal_to", agent_b, payload).await;

    // 6. Verify WebSocket receives signal
    let msg = ws_client.recv().await;
    assert!(matches!(msg, ServerMessage::Signal { .. }));
}
```
