# Step 8.0: Fix Hash Computation (CRITICAL)

## Goal

Compute proper Blake2b content hashes for entries and actions so that published data can be validated by other Holochain nodes.

## HoloHash Structure (39 bytes total)

```
[3 prefix bytes] + [32 hash bytes] + [4 location bytes]
```

**Prefixes:**
- EntryHash: `[0x84, 0x21, 0x24]`
- ActionHash: `[0x84, 0x29, 0x24]`
- AgentPubKey: `[0x84, 0x20, 0x24]`
- DnaHash: `[0x84, 0x2d, 0x24]`

---

## Entry Hash Computation

1. **Serialize entry** with MessagePack (rmp_serde format with struct_map + string_variants)
2. **Hash** with BLAKE2b-256 (32 bytes output, no key/salt/personalization)
3. **Compute DHT location** (4 bytes) from XOR-folded BLAKE2b-128 of hash
4. **Assemble**: `[0x84, 0x21, 0x24] + [32-byte hash] + [4-byte location]`

**Special case - Agent Entry:**
- Entry::Agent uses AgentPubKey hash directly (retyped with Entry prefix)

---

## Action Hash Computation

1. **Serialize action** with MessagePack using internally tagged enum format:
   ```typescript
   {
     type: "Create",  // Action variant name as string
     author: Uint8Array,  // Full 39-byte AgentPubKey
     timestamp: number,   // i64 microseconds
     action_seq: number,  // u32
     prev_action: Uint8Array | null,  // Full 39-byte ActionHash
     entry_type: { App: { zome_index: number, entry_index: number } },
     entry_hash: Uint8Array,  // Full 39-byte EntryHash
     weight: { bucket_id: 0, units: 0, rate_bytes: 0 }
   }
   ```
2. **Hash** with BLAKE2b-256 (32 bytes)
3. **Compute DHT location** (4 bytes)
4. **Assemble**: `[0x84, 0x29, 0x24] + [32-byte hash] + [4-byte location]`

---

## DHT Location Computation

```typescript
function computeDhtLocation(hash32: Uint8Array): Uint8Array {
  // BLAKE2b-128 of the 32-byte hash
  const blake128 = blake2b(hash32, null, 16);  // 16 bytes

  // XOR fold: 16 bytes → 4 bytes
  const loc = new Uint8Array(4);
  for (let i = 0; i < 4; i++) {
    loc[i] = blake128[i] ^ blake128[i+4] ^ blake128[i+8] ^ blake128[i+12];
  }
  return loc;  // Little-endian u32
}
```
